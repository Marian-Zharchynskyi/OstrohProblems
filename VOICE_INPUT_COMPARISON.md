## 📊 Детальне порівняння

### 1️⃣ Gemini Multimodal (Обрано)

**Як працює:**
```typescript
// Frontend: записує аудіо
const audioBlob = new Blob(chunks, { type: 'audio/webm' })

// Backend: відправляє до Gemini
{
  "contents": [{
    "parts": [
      { "text": "Транскрибуй це аудіо" },
      { "inline_data": { "mime_type": "audio/webm", "data": "base64..." }}
    ]
  }]
}
```

**Переваги:**
- ✅ Один API для всього (чат + голос)
- ✅ Gemini розуміє контекст проблем
- ✅ Виправляє помилки мовлення
- ✅ Не потрібна кредитка
- ✅ 1500 безкоштовних запитів/день

**Недоліки:**
- ⚠️ Трохи повільніше за спеціалізовані рішення (~2-3 сек)
- ⚠️ Ліміт на розмір аудіо (10 MB)

**Вартість:**
- Безкоштовно до 1500 запитів/день
- Для 50 користувачів по 30 повідомлень = достатньо

### 2️⃣ Google Cloud Speech-to-Text (НЕ обрано)

**Як працює:**
```csharp
// Потрібен окремий GCP проект
var request = new RecognizeRequest {
    Config = new RecognitionConfig {
        Encoding = "WEBM_OPUS",
        LanguageCode = "uk-UA"
    },
    Audio = new RecognitionAudio {
        Content = base64Audio
    }
};
```

**Переваги:**
- ✅ Найвища точність транскрипції
- ✅ Швидка обробка (~1 сек)
- ✅ Багато налаштувань (пунктуація, фільтри)

**Недоліки:**
- ❌ **Потрібна кредитна карта** (навіть для Free Tier)
- ❌ Окремий GCP проект і Billing Account
- ❌ Складна авторизація (service account JSON)
- ❌ Малий безкоштовний ліміт (60 хв/місяць)
- ❌ Після ліміту - платно ($0.006 за 15 сек)

**Вартість:**
- Безкоштовно: 60 хвилин/місяць
- Після: ~$1.44 за годину
- Для активного проекту: $50-100/місяць

### 3️⃣ Browser Web Speech API (НЕ обрано)

**Як працює:**
```typescript
// Працює прямо в браузері
const recognition = new webkitSpeechRecognition()
recognition.lang = 'uk-UA'
recognition.onresult = (event) => {
  const transcript = event.results[0][0].transcript
  sendMessage(transcript)
}
recognition.start()
```

**Переваги:**
- ✅ Абсолютно безкоштовно
- ✅ Найшвидше (real-time)
- ✅ Нуль навантаження на сервер
- ✅ Працює офлайн (на деяких пристроях)

**Недоліки:**
- ❌ **Не працює в Firefox** (без polyfill)
- ❌ Не працює в Safari на iOS (обмежено)
- ❌ Зупиняється при блокуванні екрану
- ❌ Потрібен HTTPS (крім localhost)
- ❌ Різна якість на різних пристроях

**Сумісність:**
- ✅ Chrome/Edge (Desktop + Android)
- ⚠️ Safari (Desktop - так, iOS - обмежено)
- ❌ Firefox (потрібен polyfill)

## 🏆 Висновок

### Чому Gemini Multimodal - найкращий вибір для цього проекту:

1. **Простота** - той самий API ключ, що вже використовується
2. **Безкоштовно** - не потрібна кредитка, 1500 запитів/день
3. **Розумність** - Gemini розуміє контекст міських проблем
4. **Надійність** - працює на всіх пристроях і браузерах
5. **Майбутнє** - можна легко розширити до структурованої обробки

### Коли використовувати інші підходи:

**Cloud Speech-to-Text:**
- Якщо потрібна максимальна точність
- Якщо є бюджет на GCP
- Для production з великою кількістю користувачів

**Browser Web Speech:**
- Для PWA додатків
- Якщо потрібен real-time ввід
- Для внутрішніх корпоративних систем (Chrome-only)
