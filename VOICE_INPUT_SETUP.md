# Налаштування голосового вводу

## 📝 Огляд

Голосовий ввід в AI чаті використовує **мультимодальні можливості Gemini** для нативної обробки аудіо. Це означає, що вам не потрібні додаткові API, налаштування або кредитні карти!

## 🎯 Чому Gemini Multimodal?

### ✅ Переваги:
- **Той самий API ключ** - використовується ключ від Google AI Studio
- **Безкоштовно** - в рамках безкоштовного ліміту Gemini (1500 запитів/день)
- **Без кредитки** - не потрібен Google Cloud Billing Account
- **Розумна обробка** - Gemini розуміє контекст і виправляє помилки мовлення
- **Проста інтеграція** - один endpoint для всього

### ❌ Що НЕ використовується:
- ~~Google Cloud Speech-to-Text API~~ (потребує кредитку і окремі налаштування)
- ~~Browser Web Speech API~~ (не працює на всіх пристроях)

## 🔧 Налаштування

### Крок 1: API ключ (вже налаштовано!)

Якщо у вас вже працює AI чат, то голосовий ввід працює автоматично! Використовується той самий ключ з `appsettings.json`:

```json
{
  "Gemini": {
    "ApiKey": "YOUR_GOOGLE_AI_STUDIO_KEY"
  }
}
```

### Крок 2: Перевірка моделі

Переконайтеся, що використовується модель з підтримкою аудіо:
- ✅ `gemini-2.0-flash` (рекомендовано, використовується зараз)
- ✅ `gemini-1.5-flash`
- ✅ `gemini-1.5-pro`

## 🎤 Як це працює

### Технічний процес:

1. **Frontend** (React):
   - Користувач натискає кнопку мікрофона 🎤
   - Браузер записує аудіо через `MediaRecorder API`
   - Формат: `audio/webm` (підтримується всіма сучасними браузерами)
   - Показується індикатор "Запис..." та кнопка скасування

2. **Зупинка запису**:
   - Користувач натискає кнопку зупинки 🔴
   - Показується індикатор "Розпізнаю голос..."
   - Аудіо автоматично надсилається на сервер

3. **Backend** (ASP.NET):
   - Отримує аудіофайл через `POST /chat/voice-message`
   - Конвертує в base64
   - Надсилає до Gemini для розпізнавання мови та обробки запиту

4. **Gemini API**:
   - Нативно обробляє аудіо (multimodal input)
   - Розпізнає українську мову
   - Класифікує намір користувача
   - Формує відповідь

5. **Результат**:
   - AI відповідь відображається в чаті
   - Без проміжного відображення транскрибованого тексту
   - Швидший та простіший UX

## 🌍 Підтримка мов

Gemini автоматично визначає мову з аудіо:
- 🇺🇦 Українська
- 🇷🇺 Російська
- 🇬🇧 Англійська
- Та багато інших

Не потрібно вказувати `languageCode` - модель сама розпізнає!

## ⚙️ Технічні деталі

### API Endpoint
```
POST /chat/voice-message
Content-Type: multipart/form-data
```

### Параметри запиту:
- `audioFile` - аудіофайл (WebM формат)
- Максимальний розмір: **10 MB**

### Відповідь:
```json
{
  "message": "AI відповідь на голосовий запит",
  "responseType": "Text" | "ProblemsList" | "Help" | "Error",
  "problems": [...]  // якщо є проблеми для відображення
}
```

### Gemini API виклик:
```
POST https://generativelanguage.googleapis.com/v1beta/models/gemini-2.0-flash:generateContent
```

Тіло запиту:
```json
{
  "contents": [{
    "parts": [
      { "text": "Промпт для транскрипції" },
      { 
        "inline_data": {
          "mime_type": "audio/webm",
          "data": "base64_audio_data"
        }
      }
    ]
  }],
  "generationConfig": {
    "temperature": 0.3,
    "maxOutputTokens": 512
  }
}
```

## 🐛 Усунення проблем

### "Не вдалося розпізнати голос"

**Можливі причини:**
1. Аудіо занадто тихе - говоріть ближче до мікрофона
2. Фоновий шум - знайдіть тихе місце
3. Аудіо занадто коротке (менше 1 секунди)

**Рішення:**
- Перевірте, чи працює мікрофон в інших додатках
- Спробуйте говорити чіткіше
- Переконайтеся, що аудіо записується (індикатор 🔴)

### Браузер не запитує доступ до мікрофона

**Причина:** Сайт не працює через HTTPS

**Рішення:**
- На `localhost` це працює автоматично
- На production використовуйте HTTPS

### Помилка API

**Перевірте:**
```bash
# Логи backend
dotnet run
```

Шукайте повідомлення:
```
Gemini audio processing error: {StatusCode}
```

## 💰 Вартість

### Google AI Studio (Gemini):
- **Безкоштовно**: 1500 запитів на день
- **Після перевищення**: API повертає помилку (не списує гроші!)

### Розрахунок для вашого проекту:
- Середня довжина аудіо: 5-10 секунд
- 1 аудіозапит = 1 запит до Gemini
- **1500 запитів/день** = ~50 користувачів по 30 голосових повідомлень

Для демо-проекту це більш ніж достатньо! 🎉

## 🚀 Можливі покращення

### ✅ Реалізовано:
1. **Індикатор прогресу** - показує "Розпізнаю голос..."
2. **Пряма обробка голосу** - без проміжної транскрипції в текст
3. **Підтримка української мови** - повна підтримка Gemini

### Короткострокові:
1. **Кешування** - зберігати відповіді в localStorage
2. **Попередній перегляд** - дозволити редагувати текст перед відправкою (опціонально)

### Довгострокові:
1. **Структурована обробка** - замість простої транскрипції, одразу витягувати:
   ```
   Промпт: "Витягни з аудіо: тип проблеми, адресу, опис. 
           Поверни JSON."
   ```
2. **Мультимовні промпти** - автоматично визначати мову і відповідати нею
3. **Sentiment analysis** - визначати емоційний тон (терміновість проблеми)
