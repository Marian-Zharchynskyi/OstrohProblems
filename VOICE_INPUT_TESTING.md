## 🚀 Як протестувати

### 1. Запустіть backend:
```bash
cd api
dotnet run
```

### 2. Запустіть frontend:
```bash
cd client
npm run dev
```

### 3. Відкрийте браузер:
```
http://localhost:5173
```

### 4. Протестуйте голосовий ввід:

1. Натисніть на кнопку чату (правий нижній кут) 💬
2. Натисніть на іконку мікрофона 🎤
3. Дозвольте доступ до мікрофона (якщо браузер запитає)
4. Говоріть своє повідомлення українською мовою, наприклад:
   - "Покажи топ проблеми"
   - "Які є проблеми з дорогами?"
   - "Що це за сайт?"
5. Натисніть на іконку зупинки 🔴
6. Побачите індикатор "Розпізнаю голос..."
7. AI відповідь з'явиться в чаті (без відображення тексту вашого запиту)

## 🐛 Можливі проблеми

### Проблема: "Не вдалося отримати доступ до мікрофона"
**Рішення:**
- Перевірте, чи сайт працює через `localhost` або HTTPS
- Перевірте налаштування дозволів браузера
- Спробуйте інший браузер (Chrome/Edge рекомендовано)

### Проблема: "Не вдалося розпізнати голос"
**Рішення:**
- Говоріть чіткіше та ближче до мікрофона
- Зменшіть фоновий шум
- Переконайтеся, що аудіо записується (індикатор 🔴)
- Перевірте логи backend на помилки API

### Проблема: API помилка 400/401
**Рішення:**
- Перевірте, чи правильний API ключ в `appsettings.json`
- Переконайтеся, що ключ від Google AI Studio (не GCP)
- Перевірте, чи не перевищено ліміт (1500 запитів/день)
